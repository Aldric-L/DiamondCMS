select()

core.php : 23

Fonction de simplification, pour les requettes SQL de type SELECT Cette fonction respecte les bonnes pratiques quant à l'utilisation de PDO, elle limite les failles SQL. Elle est donc a utiliser au maximum.

select( $db : mixed , $fetch : bool , $from : string , $wanted : array [, $where : array|bool = false ] [, $order_by : string|bool = false ] [, $desc : bool = false ] [, $limit : array|bool = false ] ) : array|bool
Parameters
$db : mixed
$fetch : bool

Type de récupération : fetch OU fetchall si $fetch = true alors fetch sinon fetchAll

$from : string

table SQL

$wanted : array

Elements recupérés par la requete (Syntaxe : soit un string "elem1, elem2" soit un array : array("elem1", "elem2", array("date_bdd", "format", "as")))

$where : array|bool = false

Condition(s) à effectuer dans la requete SQL, si false alors ignorée (Syntaxe : $where = array(array("nom_du_champ", "egalite", "valeur"), "OR", array("nom_du_champ", "egalite", "valeur"));)

$order_by : string|bool = false

Nom du champ sur le-quel appliqué le classement des élements par ordre croissant

$desc : bool = false

si true : Ordre décroissant des élements

$limit : array|bool = false

premier élement du tableau : minimum, deuxième : limite; false si ignoré

Fonction de simplification, pour les requettes SQL de type INSERT Cette fonction respecte les bonnes pratiques quant à l'utilisation de PDO, elle limite les failles SQL. Elle est donc a utiliser au maximum.

insert( $db : mixed , $into : string , $wanted : array , $values : array|bool ) : bool

Attention, cette fonction ne fonctionne que s'il y a correspondance parfaite entre les colonnes de $wanted, et les valeurs de $values qui y seront insérées Exemple : $wanted=array("col1","col2") et $values=array("valeur_de_la_col1", "valeur_de_la_col2")
