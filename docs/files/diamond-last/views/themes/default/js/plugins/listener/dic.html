Choisissez votre image

Option 1 : Choisir une image sur le serveur

Choisissez l'image que vous souhaitez en cliquant sur celle-ci parmi les prévisualisations.

Vous ne pouvez plus sélectionner d'images sur le serveur car vous avez opté pour une autre option. Pour autant, vous pouvez toujours annuler votre saisie pour changer d'option de choix.

Option 2 : Uploader une nouvelle image

Vous pouvez choisir une image de votre ordinateur. Celle-ci sera automatiquement choisie pour cet usage et sera ajoutée sur le serveur pour pouvoir donc être réutilisée une prochaine fois.

Attention ! L'image proposée doit obligatoirement être d'un format carré.

Image invalide, veuillez réessayer.


Option 3 : Utiliser une image déjà sur le net

Vous pouvez aussi choisir d'utiliser une image déjà sur le net, c'est-à-dire stockée sur un autre serveur que le votre. Pour cela vous devez uniquement vous munir du lien http/https de l'image.
